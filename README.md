CodeAlpha_StudentGradeTracker

A web-based Student Grade Tracker built with HTML,
CSS, and JavaScript.

Features

1. Add students with their grades
   
2. View all students in a table with grade status

3. Live stats: Total, Average, Highest, Lowest,
Passed, Failed

4. Delete individual students or clear all
   
5. Data persists using localStorage

6. Responsive design

How to Run

1. Clone or download this repository
2. Open index.html in any browser
3. No installation needed!

Tech Stack

1. HTML5

2. CSS3

3. Vanilla JavaScript
